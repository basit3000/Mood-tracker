## Installation

### 1. Clone the repo
```bash
git clone https://github.com/basit3000/mood-tracker.git
cd mood-tracker/backend
```

### 2. Install dependencies
```bash
pip install -r requirements.txt
```

### 3. Run the flask server
```bash
python app.py
```
The server will start at: http://localhost:5000
